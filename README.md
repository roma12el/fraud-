
# Insurance Risk Dashboard - PRO (Example)

This repository is a complete, ready-to-run Streamlit project for insurance risk analysis.
It includes:
- Multi-page Streamlit app (Exploration, Model Training, Risk Assessment, Dashboard)
- Utilities for preprocessing and training
- Example dataset
- Training script that creates example models
- Instructions to run and deploy to Streamlit Community Cloud

## Run locally
1. Create and activate a virtualenv
2. pip install -r requirements.txt
3. (Optional) python models/train_models.py
4. streamlit run app.py

## Deploy on Streamlit Community Cloud
- Push this repo to GitHub, then create a new app on https://share.streamlit.io pointing to `app.py` on branch `main`.
